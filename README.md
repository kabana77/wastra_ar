## About The Project

This program is used to accommodate the finished goods warehouse system, sales system, and sales invoices so that they are entered into the Account Receiveble (AR).
This application is for users and is part of a centralized system.
Of course, you have to contact the software developer to carry out technical configuration from the client side.
